ORANCHU
Study Group

✨團隊願景

透過一群會網頁前後端技術的人們一同打造屬於我們的網站，在學習的過程中累積實力、累積作品，了解沒見識過的大小知識。

針對網站進行實體架設，藉此了解其他網頁可能觸及的領域的相關知識，增長自己的見聞。

🔧運作模式

這個讀書會是 Orange 希望能夠找到一同學習網頁前後端的夥伴而舉辦的。

主要的運作模式

  • 每個人定期分享一到兩項知識(0.5~1hr/per times)，不強制與前後端有關，偶爾也可分享好笑的事物
  • 針對目標進行分工，並定期 review
  • 透過 git 進行版控，整合全部人的成果

👩‍🎓我們歡迎

我們歡迎已經對前後端有基礎知識的你加入!

針對團隊我們主要需要兩面向的夥伴...

1. Web Frontend / Web Backend

需要基本的...

  • HTML、CSS、Javascript
  • RWD 概念
  • 後端如何與前端對接(知道概念即可)

知道以下內容會很加分

  • git
  • NodeJS

2. UI/UX

需要基本的...

  • HTML、CSS、Javascript
  • RWD 概念

知道以下內容會很加分，且你進來一定會碰的設計工具

  • 閱覽各式各樣的酷炫網站，良好的設計美感
  • Figma
  • Mockplus

人數需求

  • 針對前後端開發，需求 2-4 人
  • 針對 UI/UX，需求 1-2 人

👌 如果你覺得不知道適不適合但是想試試看，可以先完成最下方說明的作業，有基礎能力都可以完成

📃完成作業

希望透過這項作業幫助你快速累積基本能力，作業請完成以下文件簡介的介面，且頁面需與文件內樣式相似率達 90 %，可不用完成 RWD，但有會更好。(介面皆為一個 HTML)

作業須完成介面請參考以下連結...

  • 介面一
  • 介面二(再等等)

完成作業有幾項限制

  • 不可使用如 bootstrap 等介面開發工具
  • 需完全手刻 HTML、CSS 來完成介面
  • 作業完成後和 Orange 簡介你的成果吧!
